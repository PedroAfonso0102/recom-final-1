# Editable Fields — RECOM

Mapa de campos que devem ser expostos via CMS/Painel Administrativo.

## 🏠 Home Page
- **Hero**: Eyebrow, Título, Subtítulo, Rótulo CTA 1, Link CTA 1, Rótulo CTA 2, Link CTA 2.
- **Destaques**: Título da seção, Listagem de processos em destaque.

## 📦 Fornecedores (Suppliers)
- **Geral**: Título da página, Descrição.
- **Entidade**: Nome, Slug, Logo (Upload), Link do Catálogo, Descrição Curta, Descrição Longa, Processos Relacionados.

## ⚙️ Processos (Processes)
- **Geral**: Título da página, Descrição.
- **Entidade**: Nome, Slug, Ícone/Imagem, Descrição Curta, Descrição Longa, Fornecedores Relacionados.

## 🎁 Promoções (Promotions)
- **Entidade**: Título, Descrição, Imagem, Data Início, Data Fim, Status.

## 📍 Configurações Globais (Site Settings)
- **Contatos**: WhatsApp, Telefone Fixo, E-mail Comercial.
- **Endereço**: Logradouro, Cidade, Estado, CEP.
- **Redes Sociais**: LinkedIn, Instagram, etc.
